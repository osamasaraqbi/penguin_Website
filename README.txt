Penguin Studio Website

Repo-ready website package.

Upload these files to the root of the GitHub repository:

- index.html
- CNAME
- .nojekyll
- assets/dashboard-ai-map.webp
- assets/penguin-logo-full.png
- assets/penguin-logo-full.webp
- assets/penguin-logo.svg
